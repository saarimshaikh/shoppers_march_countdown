# Countdown & Target Tracker

This is a single-page countdown and target tracker that counts down to 31 March 2026, 11:59 PM IST (Indian Standard Time, UTC+05:30).

Features
- Live countdown (days, hours, minutes, seconds)
- Set an original total target
- Add today's achieved amount
- Calculates remaining amount and recommended daily target based on remaining days
- Persists your target and accumulated achieved in browser localStorage

How to open
1. Open the folder `D:\shoppers_countdown` in your file explorer.
2. Open `index.html` in your browser (double-click or right-click -> Open with browser).

Notes
- If the deadline has passed, the countdown shows 00:00:00:00 and recommended daily target is adjusted accordingly.
 - No external assets are required by default.

Using the provided PNG as a background
- Place the PNG you attached to the conversation into the project at `D:\shoppers_countdown\assets\background.png`.
	- Create the `assets` folder if it doesn't exist.
	- Netlify will serve that file from the deployed site, and the page will automatically use it as the full-page background with a readable overlay.
